## SquadJS Switch Plugin
BanManager plugin for SquadJS.

### Example configuration
```json
{
    "plugin": "BanManager",
    "enabled": true,
    "discordClient": "discord",
    "channelID": "112233445566778899",
    "banMessageFormat": "Ban ID: {ban_id} | Reason: {reason} | Duration: {duration}"
}
```
